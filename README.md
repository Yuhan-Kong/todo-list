## Todo List

This is a simple todo list project built with React and Vite.  
The main purpose of this project is to practice setting up a React app using Vite and understand the basic workflow.

## Installation

```bash
# 1. Clone the repository
git clone https://github.com/MikoKong/todo-list.git
```

# 2. Enter folder and install dependencies
cd todo-list
npm install


 ## Run Development Server
 ```bash
 npm run dev

Then open http://localhost:5173 (http://localhost:5173) in your browser.
```
